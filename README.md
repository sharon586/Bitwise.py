# Python Basic Programs

## Description
This repository contains basic Python programs created as part of Python training.  
These programs help in understanding fundamental Python concepts such as operators, user input, type casting, conditional statements, and match-case.

## Topics
- Bitwise Operators
- Arithmetic Operations
- String Operations
- User Input
- Type Casting
- Conditional Statements (if-else)
- Match Case Statements

## Tools Used
- Python
- VS Code
- GitHub

## Purpose
The purpose of this repository is to practice and learn basic Python programming concepts.
